# Pizza Shop Backend

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Create a `.env` file (already scaffolded) and set your MongoDB URI and PORT.
3. Start the server:
   ```bash
   npm run dev
   ```

## Features
- Express + Mongoose
- Webhook endpoint (`/api/webhook/delivery-update`) with 5s simulated delay
- Example middleware
- Example Pizza and Order models
